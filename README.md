# VAT Calculator API

A simple ASP.NET Core Web API for calculating Value Added Tax (VAT).

## Run the project
- Ensure .NET 7 or later is installed
- Navigate to the project folder

- Run:
    Navigate the project folder (VatCalculatorApi\VatCalculatorApi) and run 
    dotnet run 
- Test:
    On the browser, navigate to 
    https://localhost:7133/swagger/index.html for https or 
    http://localhost:5022/swagger/index.html for http
